Subprocess — Subprocess management
	- This module intends to replace several older modules and functions:
		• Os.system
		• Os.spawn*
	- Subprocess.run
		• Run the command described by args. Wait for command to complete, then return a CompletedProcess instance
		• Function signature is largely the same as that of the Popen constructor
		• If capture_output is true, stdout and stderr will be captured
		• Stdout=PIPE and stderr=PIPE
		• Capture and combine both streams into one
			- Use stdout=PIPE and stderr=STDOUT
		• Timeout argument
